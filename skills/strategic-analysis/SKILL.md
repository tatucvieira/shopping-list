## What This Skill Does

Breaks down complex strategic questions into structured, actionable analysis. Used when the user faces a decision, opportunity, or threat that spans multiple dimensions (market, competitive, financial, organizational).

## When to Use

- User asks "Should we do X?" or "What do you think about X?"
- User shares a strategic initiative and needs a sounding board
- Competitor makes a move that needs assessment
- Market shift requires repositioning analysis
- Cross-entity decisions that affect V.tal + Nio + Tecto

## Methodology

### 1. Frame the Question
- Restate the decision in precise terms
- Identify: What's the real question behind the question?
- Define the decision criteria (what does "good" look like?)
- Clarify the time horizon

### 2. Map the Landscape
- **Internal:** Capabilities, constraints, resources, politics
- **External:** Market, competitors, regulation, technology trends
- **Cross-entity:** How does this affect/involve V.tal, Nio, and Tecto?

### 3. Generate Options
- At least 3 distinct options (including "do nothing")
- Each option with: description, pros, cons, risks, resource requirements
- Identify which option is the "default path" vs. the "bold bet"

### 4. Stress Test
- What has to be true for each option to work?
- What's the worst-case scenario?
- What would a smart critic say?
- Red team: argue against my own recommendation

### 5. Recommend
- Clear recommendation with confidence level
- Key assumptions that could invalidate it
- Decision reversibility assessment
- Suggested next steps and timeline

## Pitfalls to Avoid
- Analysis paralysis — don't let completeness delay action
- False precision — don't assign fake numbers to make it look rigorous
- Confirmation bias — actively seek disconfirming evidence
- Ignoring politics — the "best" strategy means nothing if it can't be executed given org dynamics

## Advancement Criteria
- **→ L2:** Successfully used 3+ times with positive user feedback
- **→ L3:** User starts requesting strategic analysis unprompted; outputs influence actual decisions
- **→ L4:** Analysis framework becomes tailored to holding-specific dynamics; consistently surfaces non-obvious insights
